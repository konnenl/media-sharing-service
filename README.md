# Media Sharing Service
REST API сервис для управления медиафайлами с аутентификацией пользователей, загрузки файлов и работы с постами.

## Стек технологий
- Python
- FastAPI
- PostgreSQL
- SQLAlchemy
- Docker
- ImageKit.io
- JWT Authentication

## Запуск проекта
```bash
docker compose up --build
